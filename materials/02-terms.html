핵심 용어·개념 (주관식·단답 대비)

교수님이 단어 위주 주관식을 내신다고 해서, 용어마다 한두 줄 굵은 핵심 정의로 정리했습니다. 검색해서 찾고, 인쇄해 오픈북에 쓰세요.

10주차 · 복합재료 / 웨어러블 / 그래핀

복합재료 Composite — 더 유용한 성능 조합을 얻기 위해 둘 이상의 소재를 병합한 신소재.
기지상 matrix — 전체 구조를 이루는 연속된 바탕상 (MMC·CMC·PMC).
분산상 dispersed phase — 기지에 분산되어 강도·인성 등 성능을 향상시키는 상(입자·섬유·구조).
혼합물의 법칙 rule of mixtures — 복합재료 탄성계수 Ec = VmEm + VpEp (V=부피분율, E=탄성계수).
라미네이트 laminate — 섬유강화 시트를 0°/90°로 적층·접착한 구조재. 장점=이방성 해소(물성 균형).
샌드위치 패널 sandwich panel — 낮은 밀도의 벌집(honeycomb) 코어를 face sheet 사이에 둔 경량·고굽힘강성 소재.
휘스커 whisker — 가늘고 긴 단결정 섬유. 알려진 가장 강한 섬유이나 고가.
이방성 anisotropy — 방향에 따라 물성이 다른 성질(정렬된 섬유). 강한 방향·약한 방향 존재.
CMC / PMC 효과 — CMC=세라믹 대비 인성·연성 증가, PMC=고분자 대비 탄성계수가 금속 수준으로 증가.
ITO Indium Tin Oxide — In+Sn+O의 세라믹계 투명전극. 변형 시 균열, 인듐(In)이 비싸 대체 소재 필요.
그래핀 graphene — 탄소가 육각격자로 이룬 원자 1개 두께(0.2nm)의 2차원 막. 높은 전자이동도·열전도도·강도, 투과율 97.7%.
동소체 allotrope — 화학조성은 같고 원자 배열이 다른 물질(그래핀·CNT·흑연·다이아몬드·풀러렌).
SAR specific absorption rate — 전자파흡수율. 인체 근접 사용 시 증가하며 거리의 제곱에 반비례(발암·두통 위해).
웨어러블 유형 — 휴대형(악세서리) · 일체형(직물/의류) · 신체부착형 · 생체이식형.

11주차 · 투명 디스플레이 / 배터리 기초

디스플레이 필름 2대 특성 — 낮은 전기저항 + 높은 광투과도. 둘은 반비례 경향.
면저항 sheet resistance — 필름 면적당 저항. 단위 Ω/sq.
AgNW 은 나노와이어 — PET 필름 위에 코팅, 와이어 접점(junction)으로 전류가 흐르는 투명전극. 많이 코팅할수록 저항↓·광투과도↓.
IEC 62175 — 플렉서블 디스플레이 기계적 시험 규격(cyclic/static bending, torsion 등). 개념정의 규격이라 시험체 크기·속도는 미정.
cyclic bending test — 반복 굽힘 시험. 폴더블폰의 사용환경 모사에 타당한 시험법.
저항변화율 고장기준 — ΔR/R₀ = 10%(0.1). 변형량 4~5%에서 도달. ※비틀림(torsion)이 굽힘보다 가혹.
1차 / 2차 전지 — 1차=충방전 불가(1회성) / 2차=충방전 반복 가능(rechargeable).
메모리 효과 memory effect — 미완전 방전 상태에서 충전 반복 시 최대 용량이 줄어드는 현상. 니켈카드뮴에 나타나고 리튬이온은 없음.
스웰링 swelling — 전해액이 산화하며 발생한 가스(C₂H₄ 등)로 배터리가 부푸는 현상. 폭발의 전단계(위험).
리튬이온전지 4대 구성요소 — 양극 · 음극 · 분리막 · 전해질.
분리막 separator — 다공성 고분자(PP) 막. 리튬이온이 기공을 통과해 양극-음극을 이동.
용량·기내반입 — 용량(Wh) = A·V (예 5Ah×3.7V=18.5Wh). 항공기 기내 반입은 100Wh 이하.

12주차 · 전극 소재 / 안전성 / 리싸이클링

양극소재 역할 — 배터리의 용량·전압을 결정(LCO·NCM·LFP 등). 활물질=active material.
NCM — Ni-Co-Mn 산화물 양극소재(전기차용).
Ni / Co / Mn 역할 — Ni=에너지밀도·주행거리, Co=부식방지·안전성(원가 20%), Mn=안전성.
음극소재(현재 주류) — 그라파이트(흑연). 충전속도·수명에 영향. (Si 합금=크랙, 리튬금속=발화)
음향방출 Acoustic Emission, AE — 손상 시 재료 내부에서 방출되는 초음파를 센서로 받아 패턴 분석해 전지 손상을 검출하는 모니터링법.
PHM — 건전성 예측 및 관리. 손상을 모니터링·진단하고 치명 시점을 예지해 관리하는 기술.
도시광산 Urban Mining — 폐가전·폐배터리 등에서 금속원소를 추출하는 것(광석 mining에 대비).
재사용 / 재활용 Reuse / Recycle — 재사용=폐배터리를 다른 용도로 전용(예: EV→ESS) / 재활용=분해해 Co·Li·Ni 금속 추출.
ESG — Environmental(환경)·Social(사회)·Governance(지배구조). 기업의 비재무적 가치 평가 요소(리싸이클링은 환경 E와 연결).

13주차 · 반도체 / 나노소재 / 3D 프린팅

인터포저 interposer — IC 칩과 PCB 사이에 삽입해 회로 폭 차이를 완충·연결하는 미세회로 기판(HBM 고밀도 연결용).
HBM — 고대역폭 메모리 반도체. 대량 데이터를 빠르게 저장해 GPU로 전송.
TSV / TGV — 실리콘관통비아 / 유리관통비아. TGV=유리 기판에 미세 구멍을 뚫고 Cu로 채워(hole+Cu filling) 전기 연결. 낮은 열팽창·미세배선.
나노소재 nano material — 1~100nm 크기의 소재. 벌크와 다른 독특한 물리·화학·전기적 특성.
나노 차원 분류 — 0D(나노분말·양자점) / 1D(나노와이어·튜브·로드) / 2D(박막·플레이트) / 3D(나노복합).
퀀텀 닷 Quantum Dot, QD — 크기 수 nm의 초미세 반도체 입자. 크기에 따라 색(빛 파장)이 변하는 양자효과(디스플레이·광학 소재).
나노 크기의 성질 변화 — 표면적/부피↑(반응성·촉매), 녹는점↓(예: 금 5nm 이하 급락), 강도·경도↑(결함 확률↓).
적층 제조 Additive Manufacturing, 3D 프린팅 — 소재를 층층이 쌓는 bottom-up 공정(절삭 top-down과 대비).
분말적층용융 PBF, Powder Bed Fusion — 분말 위에 레이저를 주사해 분말을 용융시켜 한 층씩 적층하는 방식(SLS). 금속 부품, 후가공 필요.

14주차 · 종합(과제 출제 개념)

matrix / dispersed phase — (복합재료) 연속된 바탕상 / 성능을 향상시키는 분산상. → 10주차.
스웰링 swelling — 전해액 산화 가스로 배터리가 부푸는 현상(폭발 전단계). → 11주차.
TGV Through Glass Via — 유리 기판에 미세 구멍을 뚫고 Cu로 채워 전기 연결을 구현하는 기술. → 13주차.
